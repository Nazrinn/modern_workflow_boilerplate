Boilerplate ready!

You can now use this boilerplate for your simple projects.

This boilerplate features include:

Hot reloading and module replacement
Supported by Webpack Dev Server and .
ES6 JavaScript
Supported by Babel and its Polyfill.
JavaScript minification
Supported by Webpack.
CSS minification
Suppoted by Optimize CSS Asset.
HTML minification
Supported by HTML Webpack Plugin.


This boilerplate also uses a few loaders to help Webpack process the code and assets and bundle them for production. Note that Webpack is able to process and make a base optimisation of Web JavaScript files on its own, but that's about it. This app use the following loaders:

Babel Loader
Allows Webpack to process ES6 JavaScript into ES5 Javascript.
CSS Loader and MiniCssExtractPlugin
Allows Webpack to process CSS files and create a separate CSS file for production (since Webpack does not do that on its own).
HTML Loader
Allows Webpack process to HTML files.
